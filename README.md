# GitHub Pages Website

This repository contains a simple static website with an HTML page and a separate JavaScript file.

## Files

- `index.html` — the webpage that loads in the browser
- `script.js` — the JavaScript file referenced by `index.html`

## Publish with GitHub Pages

1. Push this repository to GitHub.
2. Open the repository on GitHub.
3. Go to `Settings` > `Pages`.
4. Choose the `main` branch and the `/ (root)` folder.
5. Save, then wait a few minutes for GitHub Pages to publish.

Your site will be available at `https://<your-username>.github.io/<repository-name>/`.
